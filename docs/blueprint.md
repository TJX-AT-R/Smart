# **App Name**: DriveSmart Coach

## Core Features:

- Theory Module Lessons: Structured lessons covering essential driving theory topics, road signs, and hazard perception rules.
- Practice Question Bank: Access a large bank of multiple-choice practice questions categorized by topic, similar to the official test.
- Simulated Mock Tests: Timed mock tests designed to mimic the format, difficulty, and conditions of the official driving theory exam.
- AI-Powered Explanation Tool: Get detailed, AI-generated explanations for incorrect answers to help understand underlying concepts and reasoning.
- User Progress Tracking: Monitor performance across lessons and practice tests, highlighting strengths and areas needing improvement.
- Secure User Accounts: Allow users to register, log in, and save their personalized study progress and test results securely.

## Style Guidelines:

- Primary color: A confident and trustworthy deep blue (#236B98). This color signifies reliability and knowledge, promoting focus and a sense of calm in a learning environment.
- Background color: A subtle and airy off-white with a hint of the primary blue (#EDF5F8). This light, spacious background supports readability and reduces eye strain during extended study sessions.
- Accent color: A clear and bright aqua (#29B3B3). Used for interactive elements and key feedback, this color offers a refreshing contrast to guide user attention and signal completion or success.
- Font choice: 'Inter', a clean and highly readable grotesque sans-serif for both headlines and body text. Its neutral, objective appearance supports focus on the educational content.
- Use clear, concise, and illustrative line-based icons for navigation and feature identification, ensuring immediate comprehension.
- Implement a clean, organized, and responsive layout with ample white space, prioritizing content readability and logical flow for an optimal learning experience.
- Incorporate subtle transition animations for content loading and state changes, providing smooth user feedback without distraction.
